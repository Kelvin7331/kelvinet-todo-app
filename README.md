📝 Offline To-Do App (IndexedDB Powered)

A modern, responsive, and fully offline-capable To-Do List web application built using **HTML, CSS, and JavaScript**, powered by **IndexedDB** for persistent local storage.

---

📌 Features

- ✅ Add new tasks
- ✏️ Mark tasks as complete/incomplete
- 🗑️ Delete tasks
- 💾 Persistent storage using IndexedDB
- 📱 Fully responsive design (mobile-friendly)
- ⚡ Smooth UI interactions and animations
- 🌐 Works offline (no internet required)
- 📊 Dynamic task counter
- 🔒 Safe input rendering (prevents HTML injection)

---

🛠️ Tech Stack

- **HTML5**
- **CSS3 (Modern UI with variables & gradients)**
- **Vanilla JavaScript (ES6+)**
- **IndexedDB (Client-side database)**

---

🧠 How It Works

- The app uses **IndexedDB** to store tasks locally in the browser.
- Each task contains:
  - `id` (auto-incremented)
  - `text`
  - `completed` status
- Tasks are:
  - Loaded on page startup
  - Updated in real-time
  - Persisted even after refresh or offline usage
